Data Point Resource Help

Quick Reference

The relative path prefix /v1/ indicates that we are currently using version 1 of the API. The Mango REST api has 4 main parts Resources, Properties, Fields and Notifications.

Resource A set of URIs that return a common type of data
Property A set of URIs appended to a Resource that return the various Resources that are contained within the base Resource data type
Field Parts of the Resource data type that can be used to refine a request. These are always added as URL parameters
Notification WebSocket based push notifications for a Resources
