A performance da UI representa a responsividade da interface de usuário. Isso inclui todas as atividades que são gerenciadas pelo mecanismo de polling, como os indicadores de alarmes, updates da watch list, updates dos eventos, etc. Se o host do mango possui um CPU muito limitado, você pode diminuir a performance da UI para prevenir que o browser dos usuários consuma muito a CPU do host.

Se "Enviar erros para Serotonin" está habilitado, exceções que ocorram em seu Mango serão enviadas para a Serotonin para revisão. Se você está modificando o código fonte do Mango para seus próprios propósitos, por favor desabilite esta opção.

Use a opção "remover eventos" para especificar um tempo de vida máximo de um evento para que ele seja excluído do banco de dados. Essa exclusão removerá permanentemente o evento assim como qualquer comentário de usuário associado ao mesmo.

Clique no botão remover todos dados para permanentemente remover todos os valores de data points do banco de dados. Isto pode ser útil para dar ao seu Mango um "fresh start".
