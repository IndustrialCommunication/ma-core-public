O ID de Instância é automaticamente atribuído à instância do Mango e não deve ser alterado. É utilizado para atribuir direitos à documentação entre outras coisas. Caso você migre para uma versão comercial do Mango, você necessitará deste valor.

O tamanho do banco de dados é a quantidade de espaço atualmente consumida pelo banco de dados do sistema. A maior parte deste espaço é normalmente ocupada pelas instâncias de relatórios, valores pontos e eventos. Para minimizar este tamanho as características de descarte de cada um destes items devem ser configuradas da forma mais "agressiva possível", por exemplo remoção de TODOS os dados, eventos e relatórios que não sejam absolutamente indispensáveis.

Clique no ícone para procurar por uma nova versão do Mango. O sistema também faz essa chegagem periodicamente. Se existem upgrades disponíveis, o sistema o informará através de um evento. Este painel também mostra a versão do Mango que está em operação.
