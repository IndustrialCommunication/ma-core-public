Overview

This page provides a fast way to apply set and read permissions to data points. Data point groups can be filtered by creating filters to find desired points.

Creating a Filter

Filters are applied if the Filter State column is not set to Disabled. Multiple filters can be applied at the same time. Filters can use either % or * as wildcards, all other characters are treated as is. Using the value null will match null values, but using the value *null* will match a string that contains the text 'null'.

Each Filter row can be applied as a boolean And or a boolean Or by setting the Filter State accordingly. And operations require a point to match all of the required filter parameters. Or operators allow matching only one of the parameters.

Each filter parameter can be disabled or enabled for the current request by checking the check box in the column header. Disabling the parameter excludes it from the query entirely. To edit the value of the filter parameter, click in the cell for the corresponding filter and start typing. Hitting enter will close the editor and update the filter results. Filter cells cannot be edited if they are disabled.

Applying Permissions

Permissions can be applied by adding one or more groups separated by commas into the box and then clicking Apply permissions. To clear all permissions for the points in view click the Clear permissions button.
